Moving (8/31/21)

We have officially moved to a website, we also hope this is as good as news for you as it is for us. On the document, we do not have all the games accessible via one website, and after a while it can become pretty frusturating. While we still won´t have 100% control of all games, we will have control over a majority. This is benefitial for one main reason, they will have to block the website and not individual games. At first, this doesn´t sound like a benefit, but it is. When they block this website we can simply make a new one, and all the games will be unblocked because they are hosted on the website. Games that were not previously open-sourced on github cannot be ported as far as our knowledge goes, but we´ll look into it. Please report any issues to Xander or James, and remember, this website is a huge work in progress!
